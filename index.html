    A fun sequencer

    Subsequencely's goal is to provide an immediate, hands on music making tool that lets you try out your ideas, or stumble across new ones, without barriers. It's inspired by instruments like Korg's Volca series that give you a broad palette of creative tools that encourage exploration and experimentation.

    Subsequencely's features include

    • Play notes on the pad grid with customizable scale and layout settings.
    • Record notes into 8 32-step patterns that can be played independently, or chained together.
    • Edit patterns by hand in a step sequencer view.
    • Perform patterns from the session view with features like queueing, reversing, beat skip, and clock division.
    • More!

    Download

    The latest build is available here. It is a file called subsequencely.syx, which is a midi sysex file that you can send to your Launchpad Pro with a program like Sysex Librarian on OS X, and MIDI OX on Windows.

    To do this

    1. Plug the Launchpad into your computer.
    2. Turn it on while holding the setup button to start in bootloader mode.
    3. Choose the Launchpad as the destination in your sysex program and send the file subsequencely.syx.
    4. You will see big green letters reading "UPDATING" scroll across the Launchpad.
    5. Wait for this to finish, and the Launchpad will boot itself into Subsequencely.
    6. Now you have Subsequencely!

    Launchpad pro

    Subsequencely is a custom firmware for Novation's Launchpad Pro hardware. To install it, you simply plug your Launchpad Pro into your computer and send a sysex file to it. After that, Subsequencely can be used without a connection to a computer, by sending MIDI over the DIN port, or can be used to control software through USB.

    The Launchpad Pro is a versatile piece of hardware that offers loads of potential as a musical tool, including

    • RGB leds under every pad and button provide a clear and versatile display for many types of interfaces.
    • Velocity and pressure sensitive pads make it possible to play expressively.
    • Stand alone operation means that you don't have to tether yourself to a computer, and Subsequencely works out of the box with almost any MIDI capable device.
    • On board flash storage so that you don't lose your work in between sessions.

    Open source

    Subsequencely is built on top of the Launchpad Pro's open firmware, which means its source code is available and easy to modify. Bug reports and feature requests are encouraged on the issues page.
